Maintenance Windows API

Maintenance windows allow you to schedule service maintenance periods, during which no incidents will be created.
ResourceDescription
GET maintenance_windowsList existing maintenance windows, optionally filtered by service, or whether they are from the past, present or future.
GET maintenance_windows/:idGet details about an existing maintenance window.
POST maintenance_windowsCreate a new maintenance window for the specified services. No new incidents will be created for a service that is currently in maintenance.
PUT maintenance_windows/:idUpdate an existing maintenance window.
DELETE maintenance_windows/:idCancel or delete an existing maintenance window.
