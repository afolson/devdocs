Escalation Rules API

This API lets you access and manipulate escalation rules of an existing escalation policy.
ResourceDescription
GET escalation_policies/:escalation_policy_id/escalation_rulesList escalation rules for an escalation policy.
POST escalation_policies/:escalation_policy_id/escalation_rulesAppend a new escalation rule to the existing escalation rules.
PUT escalation_policies/:escalation_policy_id/escalation_rulesUpdates the entire collection of escalation rules.
GET escalation_policies/:escalation_policy_id/escalation_rules/:idGet details about an escalation rule.
PUT escalation_policies/:escalation_policy_id/escalation_rules/:idUpdate an escalation rule.
DELETE escalation_policies/:escalation_policy_id/escalation_rules/:idDelete an escalation rule.
