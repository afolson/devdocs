Contact Methods API

Contact methods describe the various channels (phone numbers, SMS numbers and email addresses) used to contact a user when an incident is assigned to the user.

Access and manipulate the contact methods for a user.
ResourceDescription
GET users/:user_id/contact_methodsList existing contact methods for the specified user.
GET users/:user_id/contact_methods/:idGet details for a contact method.
POST users/:user_id/contact_methodsCreate a new contact method for the specified user.
PUT users/:user_id/contact_methods/:idUpdate an existing contact method.
DELETE users/:user_id/contact_methods/:idRemove a contact method and any corresponding notification rules.
