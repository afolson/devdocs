GET users/:user_id/contact_methods/:id

Get details for a contact method.

Resource URL

GET https://<subdomain>.pagerduty.com/api/v1/users/:user_id/contact_methods/:id

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/users/PPSFHH7/contact_methods/P6W4C77"

Note: Content-type of the request must be application/json

Response Status: 200 OK 

{
  "contact_method": {
    "id": "P6W4C77",
    "label": "Work",
    "address": "5551234567",
    "user_id": "PPSFHH7",
    "type": "phone",
    "country_code": 1,
    "phone_number": "5551234567",
    "blacklisted": false
  }
}
