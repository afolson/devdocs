GET users

List users of your PagerDuty account, optionally filtered by a search query.

Resource URL

GET https://<subdomain>.pagerduty.com/api/v1/users

Parameters

This action is paginated. See the pagination documentation for details.
NameTypeRequiredDescription
queryStringNoFilters the result, showing only the users whose names or email addresses match the query
includeArrayNoArray of additional details to include. This API accepts contact_methods, and notification_rules.

Examples

Get all users with that contain john in the name or email address

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    --data-urlencode "query=john" \
    "https://acme.pagerduty.com/api/v1/users"

Note: Content-type of the request must be application/json

Get all users with contact methods and notification rules

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    --data-urlencode "include[]=contact_methods" \
    --data-urlencode "include[]=notification_rules" \
    "https://acme.pagerduty.com/api/v1/users"
