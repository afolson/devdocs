POST users/:user_id/notification_rules

Create a new notification rule for the specified user.

Resource URL

POST https://<subdomain>.pagerduty.com/api/v1/users/:user_id/notification_rules

Parameters

NameTypeRequiredDescription
start_delay_in_minutesIntegerYesNumber of minutes it will take for the notification rule to be activated (from the time the incident is assigned to the owning user) and an alert be fired.
contact_method_idStringYesThe id of the contact method

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X POST \
    -d '{"notification_rule":{"contact_method_id":"PNGKILZ","start_delay_in_minutes":4}}' \
    "https://acme.pagerduty.com/api/v1/users/PPSFHH7/notification_rules"

Note: Content-type of the request must be application/json

Response Status: 201 Created
