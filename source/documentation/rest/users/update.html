PUT users/:id

Update an existing user.

Resource URL

PUT https://<subdomain>.pagerduty.com/api/v1/users/:id

Parameters

NameTypeRequiredDescription
roleStringNoThe user's role. This can either be admin, user, or limited_user.
nameStringNoThe name of the user.
emailStringNoThe email of the user.
job_titleStringNoThe job title of the user.
time_zoneTime ZoneNoThe time zone the user is in.

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X PUT \
    -d '{"user":{"role":"admin","name":"Bart Simpson","email":"bart@example.com","job_title":"Developer"}}' \
    "https://acme.pagerduty.com/api/v1/users/PIJ90N7"

Note: Content-type of the request must be application/json

Response Status: 200 OK
