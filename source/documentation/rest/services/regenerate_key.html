  • Regenerate Key
Warning! The service's previous key will be invalidated, and existing monitoring integrations will need to be modified to use the new key!

POST services/:id/regenerate_key

Regenerate a new service key for an existing service.

Resource URL

POST https://<subdomain>.pagerduty.com/api/v1/services/:id/regenerate_key

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X POST \
    "https://acme.pagerduty.com/api/v1/services/PBIS5NX/regenerate_key"

Note: Content-type of the request must be application/json

Response Status: 201 Created
