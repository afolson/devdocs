PUT services/:id/disable

Disable a service. Once a service is disabled, it will not be able to create incidents until it is enabled again.

Resource URL

PUT https://<subdomain>.pagerduty.com/api/v1/services/:id/disable

Parameters

NameTypeRequiredDescription
requester_idStringNoThe user id of the user creating the maintenance window. This is only needed if you are using token based authentication.

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X PUT \
    -d '{"requester_id":"PP1565R"}' \
    "https://acme.pagerduty.com/api/v1/services/PIJ90N7/disable"

Note: Content-type of the request must be application/json

Response Status: 200 OK
