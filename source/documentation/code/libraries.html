PagerDuty API Libraries

A number of our users have contributed open source libraries:

Name Language Description
PDJS JavaScript Javascript library to access PagerDuty REST and event APIs, written in CoffeScript.
Pygerduty Python Dropbox Python Library for PagerDuty's v1 REST API.
PagerDuty::Connection Ruby Ruby API wrapper for the PagerDuty REST API
PagerDuty event gem Ruby A library for triggering and resolving incidents.
PagerDuty event package PHP A library for interacting with PagerDuty REST API
